# Medusa Scans

Medusa Scans is a premium manga reading platform with a beautiful anime-inspired interface. Users can redeem codes to access exclusive content.

## Features

- Beautiful anime-style UI with gradient backgrounds
- Redemption code system for premium content
- User authentication and account management
- Mobile-responsive design

## Technologies Used

- Next.js
- React
- Tailwind CSS
- Framer Motion
- TypeScript

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/bubaisarkar/medusa_scans.git
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```
